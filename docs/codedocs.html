Manage Commands¶

import personography¶

Manage command to import personography from MEP XML. Example usage:

python manage.py import_personography  /path/to/personography.xml

If a person has already been imported (based on mep_id), they will skipped when the import script is run again.

See mep.people.xml_model.Person.to_db_person() for more specifics on import logic.
